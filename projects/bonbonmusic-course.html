BonBonMusic - Course Website

專為音樂學習者打造的音樂家教媒合平台

專案簡介

BonBonMusic Course Website 是一個專為音樂學習者打造的音樂家教媒合平台，整合了課程開設、課程搜索、課程預約、課程管理、老師履歷，打破空間與時間，讓想學習音樂的同學，透過這個平台找到適合自己的音樂老師。

本專案為我第一個 UI / UX 設計實習專案，在既有 MVP 架構下，從使用者研究與問題盤點出發，重新規劃平台的資訊架構、使用流程與品牌視覺，作為後續改版的設計基礎。

我的角色

角色
UI / UX Designer

主要負責內容

  • 檢視既有 MVP 平台的使用流程與介面問題，整理學生端與老師端的核心痛點
  • 參與使用者訪談、競品研究，並產出使用者旅程地圖與流程優化方向
  • 規劃課程展示、師資介紹與報名流程的整體操作體驗
  • 建立網站的品牌 CI、視覺風格與 RWD 設計規範
  • 與共同設計夥伴協作完成 Wireframe 與高保真介面設計

共同設計
Janet Huang

核心問題

既有平台在 MVP 階段以功能快速上線為主，導致課程搜尋、報名與管理流程複雜，資訊層級混亂，學生與老師皆反映操作不易、學習成本高；同時品牌視覺與服務定位不夠清楚，也影響新使用者對平台的信任感與報名意願。

關鍵解法

以使用者研究為基礎，從學生端與老師端兩條使用情境重新梳理平台結構。透過使用者旅程地圖與流程重組，簡化課程搜尋與報名步驟，並以卡片式資訊呈現與清楚的流程指引，降低表單填寫與操作負擔；同時重新定義品牌視覺語言，建立一致的字體、色彩與介面風格，讓學習音樂的過程更輕鬆且具親和力。

成果與影響

本專案雖未實際改版上線，但完整走過從 MVP 問題盤點、使用者訪談、競品分析、使用者旅程地圖、資訊架構規劃，到品牌 CI 與介面設計的完整 UX 流程，為平台後續改版提供具體且可執行的設計方向。

對我而言，這也是奠定日後 UI / UX 設計方法與產品思維的重要實務經驗。
